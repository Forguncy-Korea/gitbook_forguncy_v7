# 그림 슬라이드(CarouselCellType)

그림 슬라이드 플러그인은 사진을 자동 또는 수동으로 전환할 수 있으며 일반적으로 홈페이지에서 사용됩니다.

### 플러그인 다운로드&#x20;

### 사용방법&#x20;

1. 플러그인을 다운로드합니다.
2. Forguncy Builder에서 설치하고 Forguncy Builder를 다시 실행합니다.

<figure><img src="../../../.gitbook/assets/image (835).png" alt=""><figcaption></figcaption></figure>

3\. 셀 영역을 선택하고, 셀 유형 "그림 슬라이드"로 설정합니다.

<figure><img src="../../../.gitbook/assets/image (814).png" alt=""><figcaption></figcaption></figure>

4\. 아래 표를 참고하여 셀 유형 설정을 합니다.&#x20;

<figure><img src="../../../.gitbook/assets/image (773).png" alt=""><figcaption></figcaption></figure>



| 설정             | 설명                                                                                                                                                                                                                                                                          |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 그림 데이터베이스에 연동  | 재생 중인 사진을 데이터베이스에서 가져오는지 여부입니다.                                                                                                                                                                                                                                             |
| 그림 정보 편집       | 전환된 사진 정보 편집:                                     ●그림 데이터베이스에 연동  : 테이블, 제목 필드, 설명 필드 및 그림 필드를 선택합니다. 테이블에 있는 이미지 필드의 데이터 유형은 이미지여야 합니다.                                                     ● 그림 데이터베이스에 미연동 : 사용자 가 그림을 클릭할 때 트리거되는 명령, 그림 제목, 그림 설명 및 그림 경로를 설정할 수 있습니다. |
| 좌우 슬라이드 버튼 표시  | 전면 및 후면 버튼을 표시할지 여부입니다. 클릭하면 이전 또는 다음 사진으로 전환됩니다.                                                                                                                                                                                                                           |
| 표시 옵션          | 표시기 버튼을 표시할지 여부입니다. 각 지침 버튼은 그림에 해당하며 지침 버튼을 클릭하면 해당 그림으로 전환됩니다.                                                                                                                                                                                                            |
| 제목 및 설명 표시     | 이미지의 제목과 설명을 표시할지 여부입니다.                                                                                                                                                                                                                                                    |
| 계속 재생          | 마지막 사진으로 전환한 후 재생을 중지할지 또는 루프 재생을 중지할지 여부입니다.                                                                                                                                                                                                                               |
| 자동 그림 슬라이드     | 사진의 자동 전환을 허용할지 여부입니다.                                                                                                                                                                                                                                                      |
| 표시 전환 간격       | 자동 그림 슬라이드가 허용되면 자동 그림 슬라이드 간격을 초 단위로 설정할 수 있습니다.                                                                                                                                                                                                                           |
| 일시 정지 옵션       | 자동 그림 슬라이드이 허용된 경우 마우스가 이미지 위에 있을 때 전환을 일시 중지하고 마우스가 떠난 후에도 전환을 계속할지 여부입니다.                                                                                                                                                                                                 |

5\. 실행하면 아래와 같이 그림 슬라이드가 나오는 것을 확인할 수 있습니다.
